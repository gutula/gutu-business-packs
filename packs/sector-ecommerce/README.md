# Sector Pack E-Commerce

E-commerce overlays for web or marketplace orders, payment timeout handling, RMA posture, and self-service returns.

## Pack Contract

- Kind: `sector-pack`
- Pack Type: `sector-template`
- Environment Scope: `sector`
- Required Plugins: `sales-core`, `inventory-core`, `accounting-core`, `crm-core`, `support-service-core`, `business-portals-core`
- Recommended Plugins: `subscriptions-core`, `analytics-bi-core`
- Capability Enhancing Plugins: `ai-assist-core`, `e-invoicing-core`
- Integration-Only Plugins: None
- Depends On Packs: `localization-global-base@^0.1.0`

- `objects/settings/defaults.json`
- `objects/workflows/default.json`
- `objects/dashboards/overview.json`
- `objects/reports/overview.json`
- `objects/automations/default.json`
- `tests/validation.json`
