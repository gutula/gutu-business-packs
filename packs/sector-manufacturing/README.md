# Sector Pack Manufacturing

Discrete manufacturing overlays for production, quality, inventory, and accounting-heavy operational flows.

## Pack Contract

- Kind: `sector-pack`
- Pack Type: `sector-template`
- Environment Scope: `sector`
- Required Plugins: `accounting-core`, `procurement-core`, `sales-core`, `inventory-core`, `manufacturing-core`, `quality-core`, `assets-core`
- Depends On Packs: `localization-global-base@^0.1.0`

- `objects/workflows/default.json`
- `objects/dashboards/overview.json`
