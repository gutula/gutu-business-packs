# Sector Pack Healthcare

Healthcare overlays for patient-facing intake, encounter support, pharmacy or consumable inventory, and regulated billing or consent posture.

## Pack Contract

- Kind: `sector-pack`
- Pack Type: `sector-template`
- Environment Scope: `sector`
- Required Plugins: `accounting-core`, `inventory-core`, `support-service-core`, `crm-core`, `hr-payroll-core`, `business-portals-core`
- Depends On Packs: `localization-global-base@^0.1.0`

- `objects/settings/defaults.json`
- `objects/workflows/default.json`
- `objects/dashboards/overview.json`
- `objects/reports/overview.json`
- `objects/automations/default.json`
